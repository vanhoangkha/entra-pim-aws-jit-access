# Changelog

## [1.0.0] - 2026-02-02

### Added
- Initial release
- Terraform module for IAM Identity Center permission sets
- Terraform module for group assignments
- Permissions boundary policy for privilege escalation prevention
- CloudWatch monitoring (optional)
- Complete documentation for SAML/SCIM setup
- Security groups configuration guide
- PIM configuration guide
- Testing and validation guide
- Troubleshooting guide
- Security best practices and checklist
- Architecture diagrams (5 diagrams)

### Security
- MFA enforcement in permission sets
- Deny statements for destructive actions
- Permissions boundary blocking IAM persistence
- IP restrictions for emergency access
